# Laplacian pyramid blending experiment

Experiment contrasting blending between images using linear fades vs laplacian pyramid blending, with laplacian pyramid blending. Laplacian pyramid blending results in a more natural blend, since it makes high frequency elements spill over to a lesser degree than low frequency elements.
